# Config（一次資料）

このディレクトリには、
『かんなぎダイアリー』制作において
実際に使用された Narrative Composer Config を収録する。

## config_v2.2_working.yml について
- 使用段階：連載初期（EP01設計時点）
- 目的：世界観・キャラクター・監視ルールを固定した上で、
        AI生成の揺らぎを抑制する
- 特徴：
  - 世界設定・政府機構・倫理構造を詳細に定義
  - キャラクターごとに speech_style / change_arc を明示
  - 評価項目（evaluation）を Config 内に内包

この Config はテンプレートではなく、
**特定プロジェクト用に最適化された実運用データ**である。
