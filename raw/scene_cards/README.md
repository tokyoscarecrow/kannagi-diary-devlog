# Scene Cards（一次資料）

このディレクトリには、
『かんなびメモランダム』制作において実際に使用した
シーンカード（Scene Card）を収録する。

## 収録方針
- 1シーン＝1ファイル
- ファイル名は原則として `EPxx_SCxx.md`
- 各カード冒頭にメタ情報（目的／段階／結果要約）を付与する

## 収録例
- EP01_SC01.yml  
  物語冒頭における世界観・ジャンル・トーン提示を目的とした導入シーン。
  事件や活躍を描かず、「管理される魔法者」という前提理解を優先している。

## テンプレート
再利用可能な型は以下にある：
- `templates/scene_card_template.md`
